# Sweet Copy App (DEMO)

מאגר הדגמה ללקוחות — **לא לייצור**.

```bash
git clone https://github.com/wegodata001-ship-it/sweetcopyapp.git
cd sweetcopyapp/wego-erp-v1
npm install
npm run dev
```
